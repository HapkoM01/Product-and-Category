# Домашнее задание 15.1

## Описание проекта
Проект для работы с классами Product (Товар) и Category (Категория товаров). Реализован полный функционал для управления товарами и категориями с защитой данных и магическими методами.

## Реализованный функционал

### Домашнее задание 14.1 (Базовые классы)
- [x] Класс `Product` с атрибутами: name, description, price, quantity
- [x] Класс `Category` с атрибутами: name, description, products
- [x] Атрибуты класса `Category`: `category_count` и `product_count`
- [x] Автоматический подсчет количества категорий и товаров

### Домашнее задание 14.2 (Инкапсуляция)
- [x] Приватный атрибут `__products` в классе `Category`
- [x] Метод `add_product()` для добавления товаров в категорию
- [x] Геттер `@property products` с форматированным выводом товаров
- [x] Приватный атрибут `__price` в классе `Product`
- [x] Геттер и сеттер для `price` с проверкой на положительное значение
- [x] Класс-метод `new_product()` для создания продукта из словаря

### Домашнее задание 15.1 (Магические методы)
- [x] `__str__` для `Product`: "Название продукта, X руб. Остаток: X шт."
- [x] `__str__` для `Category`: "Название категории, количество продуктов: X шт."
- [x] `__add__` для `Product`: сложение по формуле цена × количество
- [x] Оптимизирован геттер `products` с использованием `__str__`

### Дополнительное задание
- [x] Класс `CategoryIterator` для перебора товаров в категории
- [x] Поддержка итерации в цикле `for`

Проект использует `pytest` для тестирования.

```bash

poetry run pytest tests/test_masks_pytest.py -v

poetry run pytest --cov=src --cov-report=html
