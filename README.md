# Домашнее задание 15.1

## Описание проекта
Проект для работы с классами Product (Товар) и Category (Категория товаров). Реализован полный функционал для управления товарами и категориями с защитой данных и магическими методами.

## Реализованный функционал

### Домашнее задание 14.1 (Базовые классы)
-  Класс `Product` с атрибутами: name, description, price, quantity
-  Класс `Category` с атрибутами: name, description, products
-  Атрибуты класса `Category`: `category_count` и `product_count`
-  Автоматический подсчет количества категорий и товаров

### Домашнее задание 14.2 (Инкапсуляция)
-  Приватный атрибут `__products` в классе `Category`
-  Метод `add_product()` для добавления товаров в категорию
-  Геттер `@property products` с форматированным выводом товаров
-  Приватный атрибут `__price` в классе `Product`
-  Геттер и сеттер для `price` с проверкой на положительное значение
-  Класс-метод `new_product()` для создания продукта из словаря

### Домашнее задание 15.1 (Магические методы)
-  `__str__` для `Product`: "Название продукта, X руб. Остаток: X шт."
-  `__str__` для `Category`: "Название категории, количество продуктов: X шт."
-  `__add__` для `Product`: сложение по формуле цена × количество
-  Оптимизирован геттер `products` с использованием `__str__`

### Дополнительное задание
-  Класс `CategoryIterator` для перебора товаров в категории
-  Поддержка итерации в цикле `for`
